# 🎆 2026 大连跨年之旅 - 卡通地图

## 项目简介

这是一个专门为2026年跨年大连3天旅行规划的项目，采用**迪士尼乐园风格**的卡通地图设计，展示详细的行程安排、景点分布和旅行贴士。

## ✨ 设计特色

### 🗺️ 卡通地图
- **迪士尼风格** - 绿色草地、蓝色海洋、粉色道路
- **手绘图标** - 每个景点都有独特的emoji图标
- **交互式设计** - 点击景点查看详情，支持缩放
- **颜色编码** - 橙色Day1/蓝色Day2/紫色Day3
- **装饰元素** - 云朵、波浪、小树、指南针

### 📱 页面切换设计
- **地图页** - 迪士尼风格卡通地图
- **行程页** - 时间线卡片布局
- **贴士页** - 网格卡片实用信息
- **打卡页** - 景点打卡清单

## 🗓️ 行程概览

## 🛠️ 技术栈

- **纯HTML/CSS/JavaScript** - 无需构建工具
- **SVG绘图** - 卡通地图手绘风格
- **CSS动画** - 云朵漂浮、波浪动效
- **响应式设计** - 适配手机和桌面

## 🚀 使用方式

1. 直接用浏览器打开 `index.html`
2. 或者启动本地服务器：
   ```bash
   cd dalian_new_year
   python3 -m http.server 8888
   ```
3. 访问 http://localhost:8888
